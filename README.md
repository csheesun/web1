# tags 
```html
<strong> : 강조
<u> : 밑줄
<br> : 줄바꿈 (닫는 태그 따로 없다.) 
<p> : 단락 (닫는 태그 있음)
```
<p>
코드에서 엔터 두 번 하는 거 의미 없음
단락을 쓸거면 p tag 쓸것. 단락을 정보로서 가치있게 만들기 떄문(검색용이) 
br은 띄우고 싶은 만큼 넣으면 됨 
p 단점 : 정해져 있는 여백 만큼 벌어지니 자유도 떨어짐 

해결 : 
```html
<p style=”margin-top:40px;”>
```
</p>
